# Auto Play Chess

## Installation

1. Clone the project.
2. Open `config.json`.
3. Set your `username` and `password` chess.com
4. Run `ChessVer2.py` File.
5. Enjoy.

## Features

### 1. Nomal Mode (PVP):
  - Bullet.
  - Blit.
  - Rapid.
  - Custom.
  - Tournaments.
### 2. Hint Mode (PVP):
  Like Normal Mode but only show hint, not an automatic move.
### 3. Slove Puzzles:
  - Puzzles.
  - Puzzle Rush.
  - Puzzle Battle.
  - Daily Puzzle.
  - Custom.
### 4. Free Mode (PVE):
  Play against any computer you want.
